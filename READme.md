# Processo de Definição de Projeto – Integrador 2 (2026.1)

Este repositório reúne as propostas desenvolvidas pelos integrantes da disciplina de Integrador 2, com o objetivo de definir o projeto a ser executado ao longo do semestre.

Cada membro ficou responsável por selecionar uma ideia e elaborar um documento contendo:

- Descrição do projeto
- Proposta de MVP (Produto Mínimo Viável)
- Justificativa e relevância do problema abordado
- Estimativa de custos
- Possíveis desafios técnicos
- Estratégia de desenvolvimento

A decisão final será realizada de forma conjunta, com base nos seguintes critérios:

- **Custo** – Viabilidade financeira do projeto
- **Relevância** – Impacto e importância do problema abordado
- **Integração** – Nível de integração entre áreas (mecânica, eletrônica e software)
- **Interesse da equipe** – Engajamento dos membros com o projeto
